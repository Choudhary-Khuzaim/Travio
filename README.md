### Prerequisites
*   Flutter SDK: `^3.9.2`
*   Dart SDK: `^3.0.0`
*   An IDE (VS Code, Android Studio, or IntelliJ)

### Installation
1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/yourusername/travio.git
    ```
2.  **Navigate to Project Directory:**
    ```bash
    cd travio
    ```
3.  **Install Dependencies:**
    ```bash
    flutter pub get
    ```

    ```bash
    flutter run
    ```
